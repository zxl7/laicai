## 什么是 laicai-stock

laicai-stock 是一个基于 FastAPI 开发的企业级股票数据服务项目，提供股票行情查询、涨跌停状态查询、股票池查询等功能。

## 主要功能

- **股票行情查询**：获取单个股票的实时行情数据，包括最新价、涨跌幅、成交量等信息
- **涨跌停状态查询**：查询所有涨停或跌停的股票
- **涨停股池查询**：获取所有涨停的股票列表
- **跌停股池查询**：获取所有跌停的股票列表
- **炸板股池查询**：获取所有炸板的股票列表
- **强势股池查询**：获取强势股票池列表
